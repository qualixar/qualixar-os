## Academic Inspirations

Qualixar OS draws on established techniques from the following published research:

- **Mixture-of-Agents topology** — Inspired by Wang et al., "Mixture-of-Agents Enhances Large Language Model Capabilities" (2024). arXiv:2406.04692.
- **Byzantine Fault Tolerance (consensus protocols)** — Based on foundational work by Lamport, "The Byzantine Generals Problem" (1982) and Castro & Liskov, "Practical Byzantine Fault Tolerance" (1999).
- **POMDP-based decision making** — Informed by Kaelbling, Littman & Cassandra, "Planning and Acting in Partially Observable Stochastic Domains" (1998). Artificial Intelligence 101(1-2).
- **LightRAG (retrieval-augmented generation)** — Concepts adapted from the LightRAG framework for efficient retrieval-augmented generation.

These citations reflect theoretical foundations, not code reuse. All source code is original.
